# OoDD-WM
Out-of-Distribution Detection For Forgery Images Using Digital Watermarking

## Environments
	tensorflow 2.4
	numpy
	scikit-learn
	skimage
	
## Usage

Example:  

	python run.py

Arguments:  

	--dataset    ( 'cifar10' or 'cifar100' )
	--alpha      ( 0, 100, 200, 300, 400, 500 )
	--model      ( 'resnet' or 'densenet' )
